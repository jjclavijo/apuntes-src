Radio de curvatura

El radio de curvatura es una magnitud que mide la curvatura de un objeto geométrico tal como una línea curva, una superficie o más en general una variedad diferenciable embebida en un espacio euclídeo..

Radio de curvatura de una curva[editar]

El radio de curvatura de una línea curva o un objeto aproximable mediante una curva es una magnitud geométrica que puede definirse en cada punto de la misma y que coincide con el inverso del valor absoluto de la curvatura en cada punto:

R c ( s ) := 1 χ ( s ) {\displaystyle R_{c}(s):={\frac {1}{\chi (s)}}}

Por otro lado la curvatura es una medida del cambio que sufre la dirección del vector tangente a una curva cuando nos movemos a lo largo de ésta. Para una curva parametrizada cualquiera la curvatura y el radio de curvatura vienen dados por:[1]​

Si en lugar de un parámetro cualquiera usamos el parámetro de longitud de arco, la anterior ecuación se simplifica mucho, por resultar un vector tangente constante, y puede escribirse como:

Curvas planas[editar]

Para una curva plana cuya ecuación pueda escribirse en coordenadas cartesianas ( x , y ) {\displaystyle (x,y)\,} como x = x ( t ) ; y = y ( t ) {\displaystyle x=x(t);y=y(t)} donde t es un parámetro arbitrario, la expresión para el radio de curvatura se reduce a:

R c = [ ( d x d t ) 2 + ( d y d t ) 2 ] 3 2 | d x d t d 2 y d t 2 − d y d t d 2 x d t 2 | {\displaystyle R_{c}={\frac {\left[\left({\frac {dx}{dt}}\right)^{2}+\left({\frac {dy}{dt}}\right)^{2}\right]^{\frac {3}{2}}}{\left\vert {\frac {dx}{dt}}{\frac {d^{2}y}{dt^{2}}}-{\frac {dy}{dt}}{\frac {d^{2}x}{dt^{2}}}\right\vert }}}

En caso que pueda escribirse y = f ( x ) {\displaystyle y=f(x)\,} de tal modo que para cada punto de la curva exista un único valor de x {\displaystyle x} , entonces puede tomarse a x {\displaystyle x} como el parámetro arbitrario, y el radio de curvatura se puede calcular simplemente como:

R c = [ 1 + ( d f d x ) 2 ] 3 2 | d 2 f d x 2 | {\displaystyle R_{c}={\frac {\left[1+\left({\frac {df}{dx}}\right)^{2}\right]^{\frac {3}{2}}}{\left|{\frac {d^{2}f}{dx^{2}}}\right\vert }}}

Demostración[editar]

En primer lugar tenemos la ecuación paramétrica de la curva γ ( t ) : R → R n {\textstyle \gamma (t):\mathbb {R} \rightarrow \mathbb {R} ^{n}} de la que queremos deducir su radio de curvatura ρ {\textstyle \rho } . Ahora debemos buscar la ecuación paramétrica de la circunferencia ( g : R → R n {\displaystyle g:\mathbb {R} \rightarrow \mathbb {R} ^{n}} ) que toma el mismo valor que γ {\displaystyle \gamma } y además satisfaga que g ′ ( t ) = γ ′ ( t ) {\displaystyle g'(t)=\gamma '(t)} y g ″ ( t ) = γ ″ ( t ) {\displaystyle g''(t)=\gamma ''(t)} para cada t {\displaystyle t} fijado. Claramente el radio no depende de la posición ( γ ( t ) {\textstyle \gamma (t)} ), solo de la velocidad ( γ ′ ( t ) {\textstyle \gamma '(t)} ) y la aceleración ( γ ″ ( t ) {\textstyle \gamma ''(t)} ). A partir de dos vectores v {\textstyle v} y w {\textstyle w} solo se pueden obtener tres escalares independientes, que son: v ⋅ v {\textstyle v\cdot v} , w ⋅ w {\textstyle w\cdot w} y w ⋅ v {\textstyle w\cdot v} . Por lo tanto el radio de curvatura dependerá únicamente de los escalares | γ ′ ( t ) | 2 {\textstyle |\gamma '(t)|^{2}} , | γ ″ ( t ) | 2 {\textstyle |\gamma ''(t)|^{2}} y γ ′ ( t ) ⋅ γ ″ ( t ) {\textstyle \gamma '(t)\cdot \gamma ''(t)} .

La ecuación paramétrica general para una circunferencia en R n {\displaystyle \mathbb {R} ^{n}} viene dada por

g ( u ) = A cos ⁡ ( h ( u ) ) + B sin ⁡ ( h ( u ) ) + C {\displaystyle g(u)=A\cos {\left(h(u)\right)}+B\sin {\left(h(u)\right)}+C}

donde C ∈ R n {\displaystyle C\in \mathbb {R} ^{n}} es el centro de la circunferencia (aunque es irrelevante, por desaparecer al derivar), A , B ∈ R n {\displaystyle A,B\in \mathbb {R} ^{n}} son vectores perpendiculares le módulo ρ {\textstyle \rho } (es decir A ⋅ A = B ⋅ B = ρ 2 ∧ A ⋅ B = 0 {\displaystyle A\cdot A=B\cdot B=\rho ^{2}\land A\cdot B=0} ) y h : R → R {\displaystyle h:\mathbb {R} \rightarrow \mathbb {R} } es una función cualquiera doblemente diferenciable en t {\displaystyle t} .

Derivando

| g ′ | 2 = ρ 2 ( h ′ ) 2 g ′ ⋅ g ″ = ρ 2 h ′ h ″ | g ″ | 2 = ρ 2 ( ( h ′ ) 4 + ( h ″ ) 2 ) {\displaystyle {\begin{array}{lll}|g'|^{2}&=&\rho ^{2}(h')^{2}\\g'\cdot g''&=&\rho ^{2}h'h''\\|g''|^{2}&=&\rho ^{2}\left((h')^{4}+(h'')^{2}\right)\end{array}}}

si ahora igualamos a las derivadas correspondientes de γ {\displaystyle \gamma } obtenemos

| γ ′ 2 ( t ) | = ρ 2 h ′ 2 ( t ) γ ′ ( t ) ⋅ γ ″ ( t ) = ρ 2 h ′ ( t ) h ″ ( t ) | γ ″ 2 ( t ) | = ρ 2 ( h ′ 4 ( t ) + h ″ 2 ( t ) ) {\displaystyle {\begin{array}{lll}|\gamma '^{2}(t)|&=&\rho ^{2}h'^{2}(t)\\\gamma '(t)\cdot \gamma ''(t)&=&\rho ^{2}h'(t)h''(t)\\|\gamma ''^{2}(t)|&=&\rho ^{2}(h'^{4}(t)+h''^{2}(t))\end{array}}}

que se trata de un sistema en ρ {\textstyle \rho } , h ′ ( t ) {\textstyle h'(t)} y h ″ ( t ) {\textstyle h''(t)} que permite despejar ρ {\textstyle \rho } , obteniendo finalmente que

ρ = | γ ′ | 3 | γ ′ | 2 | γ ″ | 2 − ( γ ′ ⋅ γ ″ ) 2 {\displaystyle \rho ={\frac {|\gamma '|^{3}}{\sqrt {|\gamma '|^{2}\;|\gamma ''|^{2}-(\gamma '\cdot \gamma '')^{2}}}}} .

Radio de curvatura de superficies[editar]

Artículo principal: Geometría diferencial de superficies

Una superficie embebida en el espacio euclídeo tridimensional se caracteriza en cada uno de sus puntos por dos radios de curvatura. El centro de ambos radios de curvatura está situado sobre la recta que contiene al vector normal a la superficie. Si los dos radios de curvatura son finitos entonces se tiene:

  • En un punto elíptico de una superficie, los dos centros de curvatura están situados del mismo lado de la superficie, la curvatura gaussiana es positiva.
  • En un punto hiberbólico de una superficie, los dos centros de curvatura están situados en diferentes lados de la superficie, la curvatura gaussiana es negativa.
  • En un punto parabólico de una superficie, uno de los radios de curvatura es inifinito.
